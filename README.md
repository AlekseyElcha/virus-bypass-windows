**Программа для помощи при обходе вирусов на ОС Windows**

Преимущества:
- Продвинутый просмотр автозагрузки c возможностью её редактирования
- Мониторинг, снятие критичности и завершение процессов
- Быстрое восстановление системы (при помощи sfc и DISM)
- Восстановление шрифтов
- Снятие ограничений
- Быстрое открытие системных утилит (Групповые политики, Управление службами и др.)
- Перезапуск в безопасном режиме
- Разблокировка taskmgr и regedit
- Просмотр и удаление разделов
- и многое другое!

Совместимость:
- Windows 11 64-bit ~ 100 %
- Windows 11 32-bit ~ 100 %

- Windows 10 64-bit ~ 100 %
- Windows 10 32-bit ~ 100 %

- Windows 8.1 64-bit ~ 65 %
- Windows 8.1 32-bit (еще не тестировалось)
